| Version | Supported          |
| ------- | ------------------ |
| 1.x     | ✅                 |

## Reporting a Vulnerability

dbt Lens is a client-side tool — your data never leaves your browser. However, if you discover a security issue in the codebase (e.g., code injection, dependency vulnerability), please **do not open a public GitHub Issue**.

Instead, email: **noobigang@protonmail.com**

Please include:
- A description of the issue
- Steps to reproduce (if applicable)
- Any suggested fixes (optional)

I aim to respond within 48 hours and will keep you updated on the fix timeline.
